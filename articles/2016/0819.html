Linux MintでVimをビルドしてみた

今まではWindows環境のGvim（vim.orgが配布しているインストーラーでインストール）を使用していたんだけど，インストールしたLinuxのVimのクリップボードが使えなかった．確認してみたら

$ vim --version | grep clipboard

オプションが無効でした．これはいい機会だと思ってLinux MintにVimをソースコードからビルドしてインストールしたのでメモ．

実際にビルド

Vimについての情報は豊富で大抵の情報はすぐに見つかるので便利．

Linuxでのビルド方法

が，早速「１．必要なパッケージのインストール」でハマりました．

$ sudo apt-get build-dep vim
[sudo] password for xxx
Reading package lists...
Done Building dependency tree
Reading state information... Done
E: You must put some 'source' URIs in your sources.list

数時間ハマった挙句，最後の行のエラーが原因だったので以下の手順で解決．

  1. 画面右下の盾のマークをクリックしてUpdate Managerを起動
  2. `Edit' -> `Software source'を起動
  3. 一番下の`Enable source code repositories'をチェックして再起動．

このあとは無事参照先の方法でbuildできました．めでたし．
