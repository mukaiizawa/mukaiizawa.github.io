概要

大学では物理を専攻し、どこで間違ったのかプログラマーになった。Common LispとVimをこよなく愛する。車輪の再発明が大好きで、日々怪しいものを作成している。形式言語界隈が好きで勉強中。仕事はWeb系だけどあまり興味はない（爆）

最近は片手間に軽量プログラミング言語Parenを作成している。

シンプルを信条にしているが、気づいたらbiggerismに屈していて職場の先輩に馬鹿にされる。

言語

C

大学卒業まで一切プログラムを書いたことがなかったため、就職前に焦って勉強した言語。が、今日まで業務で使ったことはない（笑）Parenの作成に使用するため、必要に迫られて再学習した。

JavaScript

Common Lisp

最も好きな言語。趣味で多用。このブログもS式から生成されている。ピュアなCommon Lispに、自作のライブラリを使用することが多い。そのため、パッケージマネージャーや標準でないライブラリには暗い。

VimScript
